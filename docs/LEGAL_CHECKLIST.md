# Legal Checklist

このチェックリストは法的助言ではなく、公開前の確認項目です。迷う場合は専門家に確認してください。

## 公開前チェック

- [ ] 他サイトの文章をコピーしていない
- [ ] 画像、ロゴ、スクリーンショットの利用権を確認した
- [ ] 商標名、サービス名を正確に表記した
- [ ] 公式サービスと誤認される表現がない
- [ ] 価格、制限、プラン情報の公式ソースを確認した
- [ ] 確認日と公式URLを記録した
- [ ] アフィリエイトリンクや広告がある場合、必要な開示を入れた
- [ ] 個人情報や機密情報を含んでいない
- [ ] 利用規約に反する自動取得、スクレイピング、転載をしていない

## 収益化前チェック

- [ ] Google Search Consoleの流入クエリを確認した
- [ ] Google Analyticsのページ行動を確認した
- [ ] 収益化リンクの掲載位置が読者の判断を妨げない
- [ ] 利害関係や広告であることを必要に応じて明示した
- [ ] 比較記事で不利な情報も記載した
- [ ] 提携先の規約を確認した

## 高リスク表現

以下の表現は避けるか、根拠と注意書きを追加する。

- 「絶対」
- 「必ず儲かる」
- 「最安」
- 「公式」
- 「公認」
- 「完全無料」
- 「制限なし」
- 「誰でも成功」

## 記録方法

PR本文または記事メモに以下を残す。

```text
公式確認日:
確認URL:
確認した項目:
不確実な点:
```
